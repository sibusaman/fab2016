Antimony files for the above design.
While creating the above video, I used a trick to batch crop the individual .png files. I opened the files in Gimp used the selection tool to get some parameters for the command-line tool mogrify.
mogrify -crop [WIDTH]x[HEIGHT]+[X-offset]+[Y-offset]

Note the highlighted area, also note the position of the cursor at top left corner(Red). You start selecting the area you want using the 'Rectangle Selection' tool from bottom right. Then you will have the X and Y offset values (BLUE) and the width and height (RED).
so the final command would be.
mogrify -crop 1858x944+32+78 *.png
